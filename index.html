Stave Calculator

The Union Bridge Drum Co. Stave Calculator is a free and open source utility for calculating the dimensions, angles, and costs associated with the production of stave drums. If you would like to improve or make modifications to this utility, you can fork the project on GitHub.

This site requires the latest version of Google Chrome or Mozilla Firefox.
